## Dessert Oasis Coffee Roasters (DOCR) 

A streamlined mobile ordering platform for Dessert Oasis Coffee Roasters that allows customers to browse a premium menu, 
customize their drinks, and skip the line with instant order confirmation.<br/>

## Main Features

- Mobile Ordering: Browse and order from the full seasonal menu.<br/>
- Customization: Tailor drinks to specific preferences.<br/>
- Real-time Confirmation: Receive instant digital receipts and order status updates.<br/>
- Efficient Pickup: Designed to reduce wait times and improve the customer experience.<br/>
